# Bin-Packing-Problem-Solver-N26-Record
A Python repository for solving various Bin-Packing Problems. 1: What are Packing Problems?
Packing Problems are a type of geometrical optimization problem. They are considered strong NP-Complete which mean in computational complexity are the hardest of the problems to which solutions can be verified quickly. And are the involve packing various shapes or objects into various containers with maximal efficiency, and with the added rule that no part of the inner shapes may physically protrude out of the container.
These "problems" can be solved in many different ways. Methods include: Stochastic Optimization, Gradient Methods, and Geometrical Proof (though this method is quite dated). With a competent computer, these "problems" can be solved quite swiftly too! This tool is easy and simple to use: and helped me break the World-Record for the N26 Packing-Problem. Please note: This is only a proof of concept and not a "production" level solver. This was created by me, Joah-Frenzley, a Junior High School student, in my spare time!
